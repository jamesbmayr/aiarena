.api
use the API to send GET && POST requests to /api?authname={}&authpass={}&collection={}&{parameter}={value}...
• authname: use your account name
• authpass: use your password
• collection: select humans, robots, or arenas to return all items

then specify 1 or more fields to filter results, such as:
• id: get any item by its unique id
• name: get any human or robot by its name
• human.name: get all robot built by a specific human
• rules.cubes.colors: get all arenas with a specific cube color activated

or use complex queries similar to mongodb format:
• statistics.wins={$gt:10}: get humans or robots with greater than 10 wins
• name={$regex:/a/}: get all humans or robots with an "a" in its name
• $where=this.robots.length>3: get humans with > 3 robots
